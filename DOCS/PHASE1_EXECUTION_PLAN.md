# Phase 1 Execution Plan: Keygen Admin GUI

## Roadmap Overview
Phase 1 focuses on the **Core Data Flow**, establishing the foundation from authentication to machine management.

## Tasks
- **Task 1: Foundation & Project Setup** (Next.js, Shadcn UI, Sidebar/Layout)
- **Task 2: Authentication** (Login Page, Session Persistence, Credential Validation)
- **Task 3: Product & Policy Management** (Listings, Policy Editor)
- **Task 4: User Management** (User List, Create User, Profile)
- **Task 5: License Management** (Issuance Flow, Lifecycle management)
- **Task 6: Machine Management** (Activation visibility, Remote Deactivation)

## Dependencies
P0: Task 1 & 2. P1: Task 3 & 5. P2: Task 4 & 6.
